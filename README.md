# nstarter-function-compute

## 模板目录结构

```
.
├── src
│   ├── index.ts        # 函数计算业务调度入口
│   └── types.ts        # 函数计算业务 ts 类型定义
├── .env                # 环境变量定义文件，需自行按需创建
├── Funfile             # funcraft 环境配置文件，类似 Dockerfile
├── package.json        # npm 配置文件
├── README.md           # 说明文件
├── template.yml        # 函数计算服务配置文件
└── tsconfig.json       # typescript 配置文件
```
